Que site é este?

Este é um site onde poderás encontrar imensas bases de dados da plataforma Netflix, como por exemplo, filmes, séries, atores, etc.

Como acedo a uma base de dados?

Para acederes a uma base de dados basta selecionares a opção Bases de Dados na barra lateral e escolher depois a tua base de dados.

Posso fazer uma pesquisa numa base de dados?

Podes sim. As pesquisas nas bases de dados podem ser feitas acedendo primeiro à respetiva base de dados e depois preenchendo a caixa de texto de search.

Posso fazer uma pesquisa em todas as bases de dados?

Sim. Através da barra de search da Home do site podes fazer uma pesquisa que te devolverá os valores da tua pesquisa em todas as bases de dados.

O site contém um jogo?

Sim. Podes jogar tentando acertar qual dos atores participou num título. Para isso seleciona a opção jogo na barra lateral.
